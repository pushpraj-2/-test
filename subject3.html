Syllabus

1. Brief History of Computers: An overview of the development and evolution of computers from ancient times to modern day.

2. Decimal to Binary and Binary to Decimal Conversion: Methods for converting numbers between decimal and binary systems.

3. Introduction to Microsoft Word: Creating and formatting text documents using Microsoft Word.

4. Introduction to Microsoft Excel: Organizing and analyzing data using Microsoft Excel.

5. Introduction to HTML and CSS: Building and styling web pages using HTML and CSS.

6. Introduction to JavaScript: Adding interactivity to web pages using JavaScript.

Course Learning Outcomes

  • Understand the historical development and evolution of computers.
  • Convert between decimal and binary number systems.
  • Create and format documents using Microsoft Word.
  • Organize, analyze, and visualize data using Microsoft Excel.
  • Design and develop websites using HTML and CSS.
  • Implement interactive features on websites using JavaScript.
